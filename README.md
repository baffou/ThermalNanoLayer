# ThermalNanoLayer
Computes the optical properties of multilayers (dielectric and metallic), and computes the 3D temperature distribution resulting from light absorption

The code has been developed by G. Baffou & B. Rogez, in the context of a publication (a Tutorial):

Thermoplasmonics of metal layers and nanoholes
B. Rogez, Z. Marmri, F. Thibaudau, G. Baffou
APL Photonics 6, 101101 (2021)

https://aip.scitation.org/doi/10.1063/5.0057185
